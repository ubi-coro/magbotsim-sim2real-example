# MagBotSim Sim2Real Example TwinCAT3
This repository contains example code that shows how to use a control policy trained with MagBotSim in real-time to control the movers of a Beckhoff XPlanar system.
We also include the following example export scripts:
  - Stable-Baselines3 SAC policy to ONNX
  - TorchRL SAC policy to ONNX
